ColoredLevel

class ColoredLevel extends ClassicConverter

A logback converter generating colored, lower-case level names.

Used for example as:

%coloredLevel %logger{15} - %message%n%xException{5}
Annotations
@deprecated
Deprecated

(Since version 2.9.0) Use %highlight(%-5level) instead of %coloredLevel in logback config files
